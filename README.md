# Java-Mode
Este repositório reúne exercícios desenvolvidos durante meu processo de aprendizagem em Java, com foco no fortalecimento da base de programação antes do avanço para Programação Orientada a Objetos (POO).

Os exercícios abordam principalmente:

Tipagem de dados (primitivos e não primitivos)

Estruturas condicionais

Estruturas de repetição

Operações matemáticas

Manipulação de arrays e ArrayList

Lógica de programação aplicada

Entrada e saída de dados

Organização e compreensão da sintaxe Java

O objetivo deste repositório é acompanhar minha evolução prática na linguagem, consolidando conceitos fundamentais através da resolução de problemas simples e progressivamente mais complexos.

Todos os códigos foram escritos com foco em entendimento da lógica, evitando apenas a reprodução de soluções prontas.

🚀 Próximo passo: aplicação desses conceitos em Programação Orientada a Objetos (POO).
